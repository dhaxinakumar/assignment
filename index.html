    • DDL

    what is DDL command?

    Data Definition Language(DDL) commands are used to create, manipulate and modify objects in snowflake such as users, virtual ware houses,databases, schemes,tables, views, columns, functions, and stored procedures

    Advantages of DDL command

  • It stands for Data Definition Language
  • It is used to create database schema and can be used to define some contraints as well.
  • It basically defines the column(Attributes) of the table.
  • Basic command present in DDL are CREATE, DROP, RENAME, ALTER etc.

    • Tasks done using DDL command:

  • Create, alter, and drop schema objects
  • Grant and revoke privileges and roles
  • Analyze information on a table, index, or cluster
  • Establish auditing options
  • Add comments to the data dictionary

    • DML

    what is DML command

    A data manipulation language(DML) is a computer programming language used for adding (inserting), deleting, and modifying (updating) data in a database A popular data manipulation language is that of Structured Query Language (SQL) which is used to retrive and manipulae data in a relational database

    advantages of DML command

  • It stands for Data Manipulation Language.
  • It is used to add, retrieve or update the data.
  • It add or update the row of the table. These rows are called as tuple.
  • BASIC command present in DML are UPDATE, INSERT, MERGE etc.

    • tasks done using DML command

  • Change data in rows of a table, view, or snapshot
  • Change the characteristics of your session
  • Check constraints after each DML statement
  • Collect statistics about an object in the database
  • Create an explanation of the execution plan for a SQL statement
  • Defer checking of constraints until the end of the transaction
